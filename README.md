SCARLETT MARQUE
AN E-COMMERCE FASHION SITE

A fully functional, full-stack e-commerce application built with MongoDB, Express, React, and Node.js. This project features a robust order management system, real-time order tracking, and secure payment integration using Paystack with automated refund capabilities.

FEATURES
1. User Experience
2. Product Browsing: Search, filter, and view product details.
3. Shopping Cart: Add/remove items and adjust quantities.
4. Secure Checkout: Integrated with Paystack for seamless payments.
5. Order Tracking: Detailed visual stepper (Placed → Packed → Shipped → Delivered).
6. User Dashboard: Manage profile, view order history, and track active orders.
7. Cancellations & Refunds: Users can cancel orders before shipping

ADMIN CAPABILITIES
1. Product Management: Create, update, and delete products.
2. Order Management: Update order statuses (Pack, Ship, Deliver).
3. Inventory Control: Automatic stock adjustment upon order placement and cancellation.

TECH STACK
1. Frontend: React.js, Redux (State Management), React Router v6, Material UI (MUI).
2. Backend: Node.js, Express.js.
3. Database: MongoDB (Mongoose ODM).
4. Payment Gateway: Paystack API (Payments & Refunds).
5. Authentication: JWT (JSON Web Tokens).
